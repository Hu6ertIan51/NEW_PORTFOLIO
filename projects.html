EduSense

EduSense is a web-based Attendance Management System (AMS) designed to track student attendance and predict academic outcomes. Developed using Flask and integrated with machine learning, the platform supports role-based access for administrators, students, lecturers, and mentors. Its core feature is predicting whether a student will drop out, graduate, or remain enrolled based on attendance, academic performance, and enrollment data. This project was independently developed, combining web development, data analysis, and machine learning to provide actionable insights into student success.
